Preguntas frecuentes

¿Tienes dudas?

¿Necesito saber programar para instalarlo? +
Depende del plan. El plan SaaS ($59/mes) no requiere nada — nosotros lo manejamos todo. El plan Código Solo ($397) incluye una guía paso a paso detallada para cualquier sistema. El plan Código + SaaS ($597) nosotros lo instalamos por ti en 24-48 horas.
¿Cobra comisión por mis ventas? +
No. El 100% de tus ventas es tuyo. Las únicas comisiones son las de tu pasarela de pago (Stripe, PayPal, MercadoPago, etc.). Nosotros no cobramos nada adicional.
¿Funciona para otros tipos de negocio? +
Totalmente. El CRM base funciona para cualquier servicio. Coaches de vida, psicólogos, nutricionistas, academias, estudios de yoga y más ya lo usan.
¿Qué pasa si necesito ayuda con la instalación? +
El plan SaaS y el plan Código + SaaS incluyen soporte completo. El plan Código Solo incluye soporte por email por 30 días para resolver cualquier duda durante la instalación.
¿Puedo usar mi propio dominio? +
Sí. Con el plan Código Solo puedes conectar tu dominio en Vercel gratis. Con el plan SaaS y Código + SaaS nosotros lo configuramos por ti.
